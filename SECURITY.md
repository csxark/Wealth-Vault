# Security Policy

## Supported Versions

This section lists the versions of Wealth‑Vault that are actively supported with security updates.

| Version | Supported |
| ------- | --------- |
| main / latest | ✅ Supported |
| < latest | ❌ Unsupported |

We encourage using the latest version for security fixes and improvements.

---

## Reporting a Vulnerability

If you discover a security vulnerability in Wealth‑Vault, please report it **privately** so that we can investigate and address it responsibly.

### How to Report

Email the maintainers

Your report should include:

- A clear description of the vulnerability  
- Steps to reproduce the issue  
- Expected vs. actual behavior  
- Any relevant code, logs, or screenshots

This will help us assess and fix the issue more quickly.

### What to Expect

- You will receive an acknowledgment within **48 hours** of reporting.  
- We will investigate and provide regular status updates.  
- If confirmed, we will coordinate a fix and release it in the next patch release.  
- A security advisory may be published once the issue is resolved.  
- You may be credited in the advisory unless you request otherwise.

---

## Scope

This policy covers only versions listed as supported above. Reports for unsupported versions may not receive updates or a fix.

---

Thank you for helping improve the security of Wealth‑Vault!
